Deutsch
-------
Die Funktionsbibliothek für 64-Bit Betriebssysteme ist mit dem 
Zusatz "_64" bezeichnet um eine Verwechslung mit der 32-Bit Version 
zu vermeiden.

Wird die 64-Bit DLL verwendet, so muss diese von "eib7_64.dll" in "eib7.dll" 
umbenannt werden. Das selbe Verfahren gilt für die Dateien "eib7_64.lib" 
und "libeib7_64.so", die bei Verwendung in "eib7.lib" bzw. "libeib7.so" 
umbenannt werden müssen.



English
-------
The suffix "_64" was appended to the function library for
64-bit operating systems in order to avoid mistaking it
for the 32-bit version.

If the 64-bit DLL is used, it must be renamed from "eib7_64.dll"
to "eib7.dll". The same procedure applies to the files "eib7_64.lib"
and "libeib7_64.so", which must be renamed as "eib7.lib" and
"libeib7.so" if they are to be used.
